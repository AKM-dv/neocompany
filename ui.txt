════════════════════════════════════════════════════════════════
  DESIGN LANGUAGE & AESTHETIC DIRECTION
════════════════════════════════════════════════════════════════

  Overall Tone:
    Refined, modern, trustworthy. Feels like a premium service
    brand — not a marketplace. Clean but warm. Professional but
    approachable. Think "luxury home services meets editorial
    magazine layout."

  Typography System:
    Display Font  : "Clash Display" or "Syne" — for headings,
                    hero titles, section labels. Sharp, geometric,
                    high personality.
    Body Font     : "DM Sans" or "Instrument Sans" — clean,
                    legible, slightly warm. Used for paragraphs,
                    form labels, descriptions.
    Mono/Detail   : "JetBrains Mono" — used sparingly for booking
                    refs, prices, tags, status badges.

  Color Palette (Default — Editable via Admin CMS):
    --primary       : #1A1A2E   (Deep navy — trust, authority)
    --accent        : #E8622A   (Warm burnt orange — energy, CTA)
    --surface       : #F7F5F0   (Off-white cream — warmth, softness)
    --surface-dark  : #111118   (Admin panel background)
    --text-primary  : #1C1C1E
    --text-muted    : #6B6B80
    --border        : #E2E0DA
    --success       : #2E7D5E
    --warning       : #D4860B
    --danger        : #C0392B

  Spacing System:
    Base unit: 8px. Scale: 4, 8, 12, 16, 24, 32, 48, 64, 96px.

  Border Radius:
    Cards: 16px | Buttons: 10px | Tags/Badges: 6px | Inputs: 10px

  Shadows:
    Card shadow : 0 2px 16px rgba(0,0,0,0.07)
    Hover shadow: 0 8px 32px rgba(0,0,0,0.13)
    Modal shadow : 0 24px 64px rgba(0,0,0,0.18)

  Motion Principles:
    - Page load: staggered fade-up (opacity 0→1, translateY 20px→0)
      with 80ms delay between elements.
    - Hover on cards: subtle translateY(-4px) + shadow increase,
      200ms ease-out.
    - Cart drawer: slides in from right, 300ms cubic-bezier ease.
    - Buttons: scale(0.97) on press, immediate release.
    - All transitions: prefer ease-out, 150–300ms range.

  Grid System:
    Max content width: 1280px, centered.
    Column grid: 12-col with 24px gutter on desktop,
                 4-col with 16px gutter on mobile.

════════════════════════════════════════════════════════════════
  COMPONENT STATES & MICRO-INTERACTIONS
════════════════════════════════════════════════════════════════

  Buttons:
    Default → Hover (brightness +10%, translateY -1px)
    → Active/Press (scale 0.97, brightness -5%)
    → Loading (spinner inside, text fades, disabled)
    → Success (green flash for 1s on confirm actions)
    → Disabled (opacity 0.4, cursor not-allowed)

  Form Inputs:
    Default: --border outline
    Focus: --accent outline (2px), soft outer glow
    Filled: label floats up (12px, smaller)
    Error: red outline + error text slides in below (12px)
    Success: green checkmark appears right side

  Toggle Switches (Active/Inactive):
    Track: grey (off) → --accent (on)
    Thumb: white circle, slides with spring animation
    Transition: 200ms spring

  Table Rows:
    Default: transparent
    Hover: surface-2 background, 150ms ease
    Selected: --accent/8% background

  Modals:
    Entry: backdrop fades in (200ms), modal scales from 95%→100%
    + fades in (250ms).
    Exit: reverse.
    Backdrop click closes modal.
    ESC key closes modal.

  Toast Notifications (Admin):
    Appear top-right. Slide in from right.
    Auto-dismiss after 4s with progress bar.
    Types: success (green), error (red), info (blue).

  Empty States:
    Centered illustration (simple line art SVG).
    Heading: what's missing. Subtext: what to do.
    CTA button when applicable.

  Loading States:
    Tables: skeleton rows (animated shimmer).
    Cards: skeleton card blocks.
    Full page: centered spinner (--accent color).


════════════════════════════════════════════════════════════════
  RESPONSIVE BREAKPOINTS
════════════════════════════════════════════════════════════════

  Mobile   : < 640px
  Tablet   : 640px – 1024px
  Desktop  : > 1024px
  Wide     : > 1280px

  Key responsive changes:
  - Navbar → hamburger menu under 768px
  - Hero → reduced height, font size on mobile
  - Services grid → 1-col on mobile, 2-col tablet, 3-col desktop
  - Checkout → single column on mobile
  - Admin sidebar → icon-only on tablet, hidden + overlay on mobile
  - CMS/Bookings tables → horizontal scroll on mobile


════════════════════════════════════════════════════════════════
  ACCESSIBILITY NOTES
════════════════════════════════════════════════════════════════

  - All interactive elements: visible focus ring (--accent,
    2px outline, 2px offset).
  - Color contrast: all text meets WCAG AA (4.5:1 minimum).
  - Form inputs: associated <label> elements always present.
  - Images: meaningful alt text required.
  - Cart drawer: focus trap while open, ESC to close.
  - Modals: focus trap + aria-modal + role="dialog".
  - Status badges: not color-only; include text label.
  - Buttons: aria-label when icon-only.
